## 🛠️ Instalación

No requiere dependencias adicionales, solo Python 3 con tkinter.

```bash
# Clonar o descargar los archivos
python calculator.py
```

## 🚀 Uso Cyber

### Ejecutar la calculadora:

```bash
python calculator.py
```

o usando el launcher cyber:

```bash
python run_calculator.py
```

### 🎮 Controles Futuristas:

#### Teclado Cyber:
- **Números (0-9)**: Ingreso de datos
- **Operadores (+, -, *, /)**: Cálculos matemáticos
- **Enter o =**: Ejecutar cómputo
- **C**: Reset completo del sistema
- **Backspace**: Borrado selectivo
- **• (punto)**: Decimales de precisión
- **%**: Conversión porcentual

#### Interfaz Táctil:
- **CLR**: Reset total del sistema
- **⟸**: Retroceso cyber
- **±**: Inversión de polaridad
- **%**: Módulo porcentual
- **÷, ×, −, +**: Operaciones cuánticas
- **•**: Punto decimal futurista
- **=**: Ejecutor de resultados
