# 🧮 Calculadora Elegante con Flet

Una calculadora moderna y elegante desarrollada en Python utilizando la librería Flet para crear una interfaz gráfica atractiva y dinámica.

## ✨ Características

- **Interfaz moderna**: Diseño elegante con tema oscuro
- **Botones responsivos**: Efectos visuales y colores diferenciados
- **Operaciones básicas**: Suma, resta, multiplicación y división
- **Funciones adicionales**: 
  - Cambio de signo (±)
  - Porcentaje (%)
  - Borrar último dígito (⌫)
  - Limpiar todo (C)
- **Manejo de errores**: Prevención de división por cero
- **Formato inteligente**: Los números se muestran de forma optimizada

## 🚀 Instalación y Ejecución

1. **Instalar las dependencias**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Ejecutar la calculadora**:
   ```bash
   python calculadora.py
   ```

## 🎨 Diseño

La calculadora utiliza:
- **Tema oscuro** para una apariencia moderna
- **Colores diferenciados**:
  - Gris claro para funciones (C, ⌫, %, ±)
  - Gris oscuro para números (0-9, .)
  - Naranja para operadores (+, -, ×, ÷, =)
- **Botones circulares** con efectos de elevación
- **Tipografía clara** y legible

## 📱 Uso

La calculadora funciona como cualquier calculadora estándar:
1. Ingresa números haciendo clic en los botones numéricos
2. Selecciona una operación (+, -, ×, ÷)
3. Ingresa el segundo número
4. Presiona "=" para obtener el resultado

### Funciones especiales:
- **C**: Limpia toda la operación
- **⌫**: Borra el último dígito ingresado
- **%**: Convierte el número actual a porcentaje (divide por 100)
- **±**: Cambia el signo del número actual

## 🛠️ Tecnologías

- **Python 3.7+**
- **Flet**: Framework moderno para interfaces gráficas basado en Flutter
- **Diseño responsivo**: Se adapta al tamaño de la ventana
